# CSV Data Quality Audit Report

## 個人觀察(人為)

### 1. 資料中有空值

- **問題描述**：資料集中存在多個欄位的空值現象
- **影響範圍**：村里欄位發現209個空值
- **潛在影響**：影響資料完整性和後續分析準確性

### 2. 避難收容處所地址未正規化

### (1) 數字格式未統一

- **問題現象**：
    - 混用阿拉伯數字：`12-1號`
    - 混用中文數字：`十二之一號`
- **影響**：地址格式不一致，影響資料標準化

### (2) 地標而非地址

- **問題現象**：部分記錄使用地標描述而非具體地址
- **範例**：`營區位置恕不敘明`
- **影響**：降低地址精確性

### (3) 英文地址混用

- **問題現象**：包含英文地址格式
- **範例**：`No.105,Zhengxing,Jinfeng Township,Taitung County 964,TaiwanR.O.C.`
- **影響**：中英混用造成格式不統一

### (4) 其他格式問題

- **問題現象**：地址包含多餘空格、特殊字符
- **影響**：影響資料處理效率

### 3. 預計收容村里分隔未統一

- **問題現象**：
    - 使用頓號分隔：`桃山村、竹林村`
    - 使用空格分隔：`桃山村 竹林村`
- **影響**：分隔符號不統一，影響資料解析

### 4. 電話號碼格式未統一

- **問題現象**：
    - 包含特殊字符：`03-3821500#124`
    - 包含全形符號：`8712367＊710`
    - 格式混亂：`4.72E+11`
- **影響**：電話號碼格式不一致，影響聯繫資訊準確性
